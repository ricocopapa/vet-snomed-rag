# 녹음 가이드 — Track C1 합성 임상 시나리오

## 개요

5건의 합성 임상 시나리오를 사용자가 직접 발화·녹음하여 STT 인식 → SNOMED 매핑 파이프라인 평가에 사용한다. 모든 시나리오는 100% 합성 케이스로, 실환자 정보가 없다.

---

## 녹음 환경

| 항목 | 권장 사양 |
|---|---|
| 장소 | 조용한 실내 (배경 소음 최소화) |
| 마이크 | 스마트폰 내장 또는 이어폰 마이크 허용 |
| 거리 | 마이크와 입 거리 약 15~30cm |
| 배경 음악 | 금지 |
| 에어컨/선풍기 소음 | 최소화 권장 |

---

## 파일 포맷

| 항목 | 사양 |
|---|---|
| 권장 포맷 | m4a 또는 wav |
| 샘플링 레이트 | 16kHz 이상 |
| 채널 | 모노 또는 스테레오 모두 허용 |
| 비트레이트 | 128kbps 이상 권장 |

## 파일명 규칙

```
scenario_1.m4a    (안과: 고양이 녹내장 의심)
scenario_2.m4a    (소화기: 개 급성 위장염)
scenario_3.m4a    (정형외과: 개 슬개골 탈구)
scenario_4.m4a    (피부과: 고양이 피부사상균증)
scenario_5.m4a    (종양학: 개 체표 비만세포종 의심)
```

wav 포맷 사용 시 `.m4a` → `.wav`로 변경.

---

## 발화 가이드

### 기본 원칙

1. **자연스러운 임상 구어체** — 실제 수의사가 SOAP 차트를 구술하는 톤으로 발화
2. **속도** — 평상시 대화 속도. 너무 느리거나 빠르지 않게
3. **숫자 발화** — "32밀리미터에이치지", "39.2도", "2주" 등 자연스럽게 읽기
4. **약어 발화** — "MPL" → "엠피엘" 또는 "슬개골 내측 탈구"로 풀어서 발화
5. **한 번에 연속 발화** — 중간에 긴 정지 없이 전체 스크립트를 한 호흡으로

### 목표 발화 시간

| 시나리오 | 도메인 | 목표 시간 |
|---|---|---|
| scenario_1 | 안과 (OPHTHALMOLOGY) | 약 60초 |
| scenario_2 | 소화기 (GASTROINTESTINAL) | 약 75초 |
| scenario_3 | 정형외과 (ORTHOPEDICS) | 약 65초 |
| scenario_4 | 피부과 (DERMATOLOGY) | 약 70초 |
| scenario_5 | 종양학 (ONCOLOGY) | 약 65초 |

### 재녹음 기준

아래 경우 재녹음 권장:
- 전체 발화 시간이 30초 미만 또는 90초 초과
- 중간에 3초 이상 침묵 발생
- 배경 소음으로 핵심 임상 수치가 묻힌 경우
